The tech stack

The tech stack is broken down into a few parts, as shown at the image below.

Infrastructure

The server is hosted on a DigitalOcean server (Droplet) running Ubuntu 20.04, whereas the code is hosted on GitHub.

We separate our development into a few parts, to enable productive development.

Deployment Lifecycle

Frontend

  1. The developer first commits and pushes the code to GitHub.
  2. A cron job, running on the DigitalOcean Droplet will pull the latest changes, compile codes and copies them to the Apache server directory.
  3. The new page is served for new web page requests.

Backend

  1. The developer first commits and pushes the code to GitHub.
  2. A cron job, running on the DigitalOcean Droplet will pull the latest changes and copies them to the php directory.

Database

All changes are done directly on the database.

Usage Lifecycle

  1. The user will first encounter the compiled front end page written in Angular.
  2. For external API requests, it will directly redirect to the relavant server; for internal API requests, it will send to the internal PHP server for processing.
  3. If required, the PHP server will connect with the MySQL server to get data. Else it directly responds.
  4. Angular intercepts the data and presents it at the relavant location.
